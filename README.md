# The Playlist Page

## Getting started

1. Install yarn
    ```
    $ curl -o- -L https://yarnpkg.com/install.sh | bash
    ```
2. Install dependencies
    ```
    $ yarn install
    ```
3. Run the development server (includes live reloading)
    ```
    $ yarn serve
    ```
4. Open the page
    ```
    $ open http://localhost:8080
    ```
